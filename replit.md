# New Sanical Website

## Overview
A Next.js website for a plumbing and heating service company (Sanical) based in Charleroi, Belgium. The site is in French and showcases their services with a modern design.

## Tech Stack
- Next.js 15.x
- React 19
- TypeScript
- Tailwind CSS 4.x

## Project Structure
- `app/` - Next.js app router pages
- `components/` - React components
- `public/` - Static assets (images, icons)

## Development
The dev server runs on port 5000:
```
npm run dev
```

## Build & Deploy
- Build: `npm run build`
- Start production: `npm run start`

Deployment is configured for autoscale with Next.js production server.
